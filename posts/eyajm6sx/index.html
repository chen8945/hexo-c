Azure Active Directory(AAD)应用密码自定义到期时间

发表于2022-06-16|更新于2022-06-16|好记性不如烂笔头
|字数总计:255|阅读时长:1分钟

网站

Azure Active Directory应用注册: https://portal.azure.com/#view/Microsoft_AAD_IAM/ActiveDirectoryMenuBlade/~/RegisteredApps
Microsoft Graph: https://developer.microsoft.com/zh-cn/graph/graph-explorer

步骤

Microsoft Graph需要使用全局管理员登录，否则会提示需要管理员授权。

  1. 进入Azure Active Directory应用注册界面新注册一个应用或者点击已有应用

  2. 复制对象ID

  3. 进入Microsoft Graph将请求方法改为POST，并将请求地址为https://graph.microsoft.com/v1.0/applications/{上一步复制的对象ID}/addPassword，替换{上一步复制的对象ID}

  4. 修改权限

  5. 添加请求正文，此处示例到期时间为2099年12月31日，displayName为密码说明，endDateTime为密码到期时间，请按照自己的需求更改

  1. 点击右上角运行查询，响应预览内容中secretText即是需要的密码

  2. 返回应用界面点击证书和密码确认是否成功
