Markdown语法学习笔记

发表于2021-08-09|更新于2021-08-19
|字数总计:552|阅读时长:2分钟|阅读量:

Markdown 标题语法

　#　的数量代表了标题的级别,添加三个　#　表示创建一个三级标题。

注意事项

为了在不同程序之间的兼容性考虑，请用一个空格在　#　和标题之间分隔。

Markdown 强调语法

加粗

在需要加粗部分的两侧各添加两个　*　或者两个　_　,文字与符号之间不需要空格。


如下所示：

This is my blog.
This is my blog.

倾斜

在需要倾斜部分的两侧各添加一个星号,文字与符号之间不需要空格。


如下所示：

This is my blog.

Markdown 引用语法

段落前添加　>　,可渲染多行文字。


如下所示：

This is my Blog.

引用嵌套

按顺序依次添加　>　。


如下所示：

This is my blog.

This is my blog.

This is my blog.

Markdown 列表语法

无序列表

每个列表项前面添加　破折号 ( - )、星号 ( * ) 或加号 ( + )　。


如下所示：

  • This is my blog.
  • This is my blog.
  • This is my blog.

无序列表嵌套

按顺序依次缩进制表符　破折号 ( - )　，其他符号同理。


如下所示：

  • This is my blog.
    • This is my blog.
      • This is my blog.

Markdown 代码语法

将所要表示的代码使用　反引号 ( ` )　包含起来。


如下所示：
This is my blog.
This is my blog.

代码块

将　反引号 ( ``` )　补充在在代码块的上下两行，将其包含起来。
在上方　反引号　后可填写代码语言类型，以高亮代码显示规范。

Markdown 分隔线语法

要创建分隔线，请在单独一行上使用　三个或多个星号 (*)、破折号 (—-) 或下划线 (_)　，并且不能包含其他内容。


如下所示：

This is my blog.

This is my blog.

This is my blog.

注意事项

为了兼容性，请在分隔线的前后均添加空白行。
